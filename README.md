# High-Nutrition Seal Proposal

This repository contains the draft proposal for a voluntary 'High-Nutrition' front-of-pack food seal, including nutrient criteria and the manufacturer application process.

## Purpose

The goal is to provide a simple, quick visual cue for consumers to identify healthier options. This initiative is led by an inter-departmental group focused on public health nutrition.

## Contents

- Nutrient criteria for qualifying products
- Manufacturer application process and form
- Consumer FAQ
- Research on international examples
